Tools for Front End Developer

Font Awesome

Font Awesome adalah kumpulan fon dan ikon yang berbasis pada CSS dan Less. Font Awesome dibuat oleh Dave Gandy untuk digunakan dengan Bootstrap, dan kemudian dibundel ke BootstrapCDN.

Go to Link

W3Schools

W3Schools adalah situs web developer informasi, dengan tutorial dan referensi yang berkaitan dengan topik pengembangan web seperti HTML, CSS, JavaScript, PHP, SQL, dan JQuery.

Go to Link

Smooth Shadow

Dengan trik CSS sederhana, kita bisa mendapatkan kontrol yang sempurna atas bagaimana bayangan dirender, dan menciptakan efek 3D yang lebih kaya dan lebih realistis.

Go to Link

Animate.css

Animate.css adalah perpustakaan animasi lintas-browser siap pakai untuk digunakan dalam proyek web Anda. Bagus untuk penekanan, halaman beranda, penggeser, dan petunjuk panduan perhatian.

Go to Link

GitHub

GitHub adalah layanan hos web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem kendali versi Git dan layanan hosting internet. Hal ini banyak digunakan untuk kode komputer.

Go to Link

Color Hunt

Temukan palet warna pilihan terbaru dari Color Hunt. Dapatkan inspirasi warna untuk proyek desain dan seni Anda.

Go to Link

Google Fonts

Google Fonts adalah layanan font web milik Google. Ini termasuk keluarga font sumber terbuka dan gratis, direktori web interaktif untuk menjelajahi perpustakaan, dan API untuk menggunakan font melalui CSS dan Android.

Go to Link

Bootstrap

Bootstrap adalah kerangka kerja CSS yang sumber terbuka dan bebas untuk merancang situs web dan aplikasi web. Kerangka kerja ini berisi templat desain berbasis HTML dan CSS untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainnya, serta juga ekstensi opsional JavaScript.
